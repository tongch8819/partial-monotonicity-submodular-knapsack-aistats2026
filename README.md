# Partial Monotonicity for Submodular Maximization with a Knapsack Constraint

This repository contains the official implementation for the paper **"Partial Monotonicity for Submodular Maximization with a Knapsack Constraint"**, accepted at the 29th International Conference on Artificial Intelligence and Statistics (**AISTATS 2026**).

This codebase includes the implementation of our proposed algorithms and the scripts necessary to reproduce the empirical results presented in the paper.

To run the main experiments described in the paper:

```bash
cd experiments
# Basic execution for two applications
python run.py all

# Execution for individual applications
python run.py movie
python run.py influence
```
